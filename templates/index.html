NOAA Severe Weather Data 2010-2016

This is a visual analysis of hail storms, tornados, and severe wind events that occured in the United States of America. Data was retrieved from Severe Weather data from the NOAA Storm Prediction Center converted to a Geojson file or SQLAlchemy db. To plot the data, we used Leaflet, D3 and Plotly and deploy the website in Heroku using Flask and MongoDB.
